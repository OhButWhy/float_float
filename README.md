# float_float
Программа для расчёта суммы значений синуса в узлах равномерной сетки

Тип массива, в котором хранятся значения синуса можно выбрать во время сборки

Используйте флаг -DUSE_FLOAT=true, если хотите выбрать тип float
По умолчанию массив имеет тип данных double

Результат для float: 7.27596e-06
Результат для double: 3.81905e-14
